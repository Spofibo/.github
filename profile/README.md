# Spofibo

Spofibo is an independent digital holding and operating group focused on durable ventures and selective technical advisory.

This org hosts shared infrastructure, cloud and application platform code we use to keep our ventures, brands and partner systems robust and easy to operate.

## What lives here

- Terraform modules for AWS, Azure, GCP, Cloudflare and related infrastructure
- Ansible roles and automation for servers and platforms
- Supporting tooling for our ventures and specialist brands

For selected projects, we also offer technical advisory services around platforms, infrastructure and modernisation.  
To learn more or get in touch, visit https://spofibo.com.
